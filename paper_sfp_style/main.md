# Support Density Matters: Density-Guided Feature Tracing for Training-Free Open-Vocabulary Semantic Segmentation

## Abstract

Training-free open-vocabulary semantic segmentation benefits from frozen
vision-language models, but dense prediction remains unstable when patch-level
visual evidence is unevenly distributed. We observe that many errors arise not
only from semantic mismatch, but from **support density imbalance**: boundary
patches, thin structures, and small objects often lack enough reliable neighbors
on the visual token manifold. To address this issue, we propose **DeFT**,
Density-guided Feature Tracing, a deterministic and training-free framework.
DeFT estimates token support density, traces reliable graph and local evidence
to low-support regions, and applies structure-conserving multi-scale
recalibration before text-image similarity. Experiments follow the same
training-free open-vocabulary segmentation protocol and use a frozen
CLIP-based runtime. The final DeFT row in the tables should be filled after
running the provided scripts.

## 1. Introduction

Training-free open-vocabulary semantic segmentation (OVSS) aims to segment
arbitrary categories without optimizing a task-specific segmentation model.
Frozen vision-language models provide strong semantic alignment, but their
image-level pretraining objective does not guarantee spatially coherent dense
features. In practice, prediction confidence is often concentrated in a few
high-support regions, while ambiguous boundaries, thin parts, and small objects
remain weakly supported.

This paper studies this issue from the perspective of **support density**. A
visual token is reliable for dense prediction when it is supported by a compact
set of semantically related tokens and by local spatial evidence. When this
support is sparse, direct text-image similarity can produce fragmented masks.
Therefore, rather than introducing learnable decoders, DeFT performs
training-free feature tracing on the frozen visual tokens.

DeFT contains three components. First, it estimates support density using the
top-k visual-token neighborhood. Second, it performs density-guided evidence
tracing from graph neighbors and local spatial windows. Third, it applies
structure-conserving multi-scale recalibration to recover continuity in
low-support regions. The refined tokens are then compared with text embeddings
using the original cosine-similarity classifier.

Contributions:

1. We formulate support density imbalance as a key obstacle in training-free

## 2. Related Work

### Open-Vocabulary Semantic Segmentation

OVSS extends semantic segmentation to arbitrary language-defined categories.
Training-based methods adapt vision-language features with extra supervision,
while training-free methods reuse frozen image-text representations. DeFT
belongs to the training-free family.

### Dense Transfer from Vision-Language Models

CLIP-like models learn strong image-text alignment, but their patch features are
not optimized for dense prediction. Existing methods improve dense transfer with
feature correction, prompt design, attention refinement, or post-processing.
DeFT is complementary: it operates on frozen visual tokens and uses support
density to decide where structure should be recovered.

### Structure Recovery without Training

Graph propagation and spatial smoothing are common tools for dense prediction.
Unlike trainable graph modules, DeFT uses a fixed similarity graph and
deterministic local aggregation, preserving the training-free protocol.

## 3. Methodology

Let `X in R^{N x C}` be frozen visual patch tokens, where `N = H x W`, and let
`T in R^{K x C}` be text embeddings for `K` candidate classes. DeFT refines
`X` before dense classification:

```text
X_hat = DeFT(X)
Y = X_hat @ T.T
```

### 3.1 Overview

The whole framework follows the frozen-runtime inference pipeline. Given an
image, the visual encoder extracts patch tokens. DeFT refines these tokens with
support-density tracing. The refined tokens are normalized and compared with
text embeddings to obtain segmentation logits.

### 3.2 Support Density Estimation

For each normalized token `x_i`, DeFT computes cosine similarity with all other
tokens and selects its top-k neighbors. The support density is the average
positive similarity over these neighbors. A density gate is generated by a
sigmoid function so that low-density tokens receive stronger refinement.

### 3.3 Density-Guided Evidence Tracing

DeFT gathers two sources of evidence:

- graph evidence from semantically related top-k tokens;
- local evidence from spatial neighborhoods on the feature map.

The two sources are mixed with fixed weights. A density gate determines how much
each original token should be replaced by traced evidence. High-support tokens
remain close to their original representation, while low-support tokens borrow
reliable evidence from their neighbors.

### 3.4 Structure-Conserving Recalibration

After tracing, DeFT applies deterministic multi-scale residual recalibration.
Average-pooled residuals from multiple kernel sizes are injected more strongly
into low-density regions. This step improves continuity while preserving the
original cosine-similarity interface with text embeddings.

## 4. Experiment

### 4.1 Experimental Settings

**Datasets.** Following the reference protocol, experiments cover PASCAL VOC,
PASCAL Context, COCO, ADE20K, and Cityscapes. The table columns are grouped into
with-background datasets (`V21`, `PC60`, `Object`) and without-background
datasets (`V20`, `PC59`, `Stuff`, `ADE`, `City`).

**Implementation Details.** We use a frozen CLIP ViT-B/16 runtime unless
otherwise specified. Images are resized with a short side of 336 pixels, and
sliding-window inference uses a 224 x 224 crop with a 112 x 112 stride. DeFT
uses fixed hyperparameters across datasets: `k=12`, density threshold `0.46`,
trace strength `0.72`, graph weight `0.65`, local weight `0.35`, and
multi-scale kernels `{3,5,7}`.

## 5. Conclusion

We present DeFT, a training-free density-guided feature tracing framework for
open-vocabulary semantic segmentation. DeFT addresses support density imbalance
by estimating token reliability, tracing graph-local evidence, and applying
structure-conserving recalibration. The framework preserves the frozen
vision-language runtime while improving dense spatial predictions. After the
